### **Installation**
```bash
# Run all database scripts in order
psql -d your_database -f 01_create_tables.sql
psql -d your_database -f 02_create_indexes.sql
psql -d your_database -f 03_sample_data.sql
psql -d your_database -f 04_supabase_setup.sql
```

### **Supabase Setup**
```bash
# Copy environment template
cp ../backend/env.example ../backend/.env

# Configure Supabase credentials
SUPABASE_URL=your_supabase_project_url
SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
```

## 🚀 **Development Workflow**

### **Local Development**
```bash
# Start local PostgreSQL
docker run --name webshop-postgres -e POSTGRES_PASSWORD=consciousness -p 5432:5432 -d postgres:14

# Run database setup
psql -h localhost -U postgres -d postgres -f 01_create_tables.sql
psql -h localhost -U postgres -d postgres -f 02_create_indexes.sql
psql -h localhost -U postgres -d postgres -f 03_sample_data.sql
```

### **Supabase Development**
```bash
# Install Supabase CLI
npm install -g supabase

# Initialize Supabase project
supabase init

# Start local Supabase
supabase start

# Run migrations
supabase db reset
```

### **Production Deployment**
```bash
# Deploy to Supabase
supabase db push

# Deploy functions
supabase functions deploy

# Deploy storage policies
supabase storage deploy
```
