**🔧 Development Commands**: Always run from `WebShop-Implementation/` directory:
```bash

npm run dev:backend  # Start only Node.js backend
```

## 🚀 Quick Start

### **Prerequisites**
- Node.js 18+ 
- npm 8+
- Git

### **Installation**
```bash

git clone <repository-url>
cd WebShop-Implementation  # ⚠️ Always work from this directory!
```

### **Development Servers**
- **Frontend**: http://localhost:4000 (React + TypeScript)
- **Backend**: http://localhost:3001 (Node.js + TypeScript)

The frontend uses `create-react-app` which doesn't have a `dev` script by default. Use:
```bash

```

Or from the root directory:
```bash
cd WebShop-Implementation
```


## 🔧 Troubleshooting

### **Common Issues**

#### **"Missing script: dev" Error**
```bash
# If you get "Missing script: dev" error:
cd WebShop-Implementation/frontend
npm start  # Use 'start' instead of 'dev'
```

#### **"Could not read package.json" Error**
```bash
# Make sure you're in the correct directory:
cd WebShop-Implementation  # ⚠️ Always work from here!
npm run dev
```

#### **Frontend Won't Start**
```bash
# Clear cache and reinstall:
cd WebShop-Implementation/frontend
rm -rf node_modules package-lock.json
npm install
npm start
```

#### **Backend Connection Issues**
```bash
# Check if backend is running:
curl http://localhost:3001/health

# Restart backend:
cd WebShop-Implementation/backend
npm run dev
```
