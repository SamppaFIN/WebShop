# WS-003 · NextAuth.js -konfiguraatio

**Vaihe:** 2 — Auth | **Riippuvuudet:** WS-000 | **Arvioitu kesto:** 12 min

## Kuvaus
Konfiguroi NextAuth.js v5 Prisma-adapterilla. Google OAuth + Credentials (email/salasana). Suojaa reitit middlewarella.

## Hyväksymiskriteerit

1. `src/auth.ts`: PrismaAdapter, GoogleProvider, CredentialsProvider (email+bcrypt), callbacks (jwt, session → user.id)
2. `src/app/api/auth/[...nextauth]/route.ts`: `export { GET, POST } from '@/auth'`
3. `src/middleware.ts`: suojaa `/checkout`, `/orders`, `/account` → ohjaa `/login`
4. `src/actions/auth.ts`: Server Actionit `signIn`, `signUp`, `signOut`
5. `src/components/providers/SessionProvider.tsx`: Client-komponentti
6. `src/lib/prisma.ts`: Prisma-client singleton
7. Google OAuth tunnukset `.env.local`:ssa
8. Kirjautuminen Googlella ja email/salasanalla toimii

## Tiedostot

```
src/auth.ts
src/app/api/auth/[...nextauth]/route.ts
src/middleware.ts
src/actions/auth.ts
src/components/providers/SessionProvider.tsx
src/lib/prisma.ts
.env.local    ← AUTH_*-muuttujat
```

## Verifiointi
1. `/login` → Google-nappi ohjaa Googleen
2. Rekisteröidy email/salasanalla
3. Kirjaudu → `/account` näkyy
4. Kirjaudu ulos
