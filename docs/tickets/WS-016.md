# WS-016 · LoginPage

**Vaihe:** 5 — Sivut | **Riippuvuudet:** WS-012, WS-003 | **Arvioitu kesto:** 10 min

## Kuvaus
Kirjautumis- ja rekisteröitymissivu. Google OAuth + email/salasana. Zod-validaatio.

## Hyväksymiskriteerit

`src/app/login/page.tsx`:
1. Keskitetty kortti `max-w-md`
2. shadcn/ui `Tabs`: "Kirjaudu" / "Luo tili"
3. Google OAuth -nappi: "Jatka Googlella" → `signIn('google')`
4. LoginForm: email + salasana, Zod-validaatio, virheenkäsittely, loading-spinner
5. SignUpForm: nimi + email + salasana + vahvistus, Zod-validaatio, `signUp` Server Action
6. Onnistuminen → `redirect('/')`

## Tiedostot

```
src/app/login/page.tsx
src/components/auth/LoginForm.tsx
src/components/auth/SignUpForm.tsx
```
