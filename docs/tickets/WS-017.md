# WS-017 · CheckoutPage + Stripe

**Vaihe:** 5 — Sivut | **Riippuvuudet:** WS-005, WS-012, WS-003 | **Arvioitu kesto:** 15 min

## Kuvaus
Kassasivu: toimitusosoite + yhteenveto + Stripe Checkout -maksu. Webhook.

## Hyväksymiskriteerit

`src/app/checkout/page.tsx`:
1. Suojattu reitti (middleware)
2. **Vaihe 1 — Osoite:** lomake (nimi, osoite, kaupunki, postinumero, maa), Zod-validaatio
3. **Vaihe 2 — Yhteenveto:** tuotteet, hinnat, yhteensä, toimitusosoite
4. **Vaihe 3 — Maksu:** "Maksa" → `POST /api/checkout` → Stripe Checkout Session → ohjaa Stripeen

`src/app/api/checkout/route.ts`:
- Luo Stripe Checkout Session: `mode: 'payment'`, `line_items`, `success_url`, `cancel_url`

`src/app/api/webhooks/stripe/route.ts`:
- Käsittelee `checkout.session.completed`: luo tilauksen, vähentää varastoa, tyhjentää cartin

`src/lib/stripe.ts`:
- `export const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!)`
