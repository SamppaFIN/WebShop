# WS-005 · Zustand cart store

**Vaihe:** 3 — State | **Riippuvuudet:** WS-002 | **Arvioitu kesto:** 8 min

## Kuvaus
Luo Zustand-store ostoskorille. Persistointi localStorageen.

## Hyväksymiskriteerit

`src/store/cartStore.ts`:
- `items: CartItem[]`
- `addItem(productId, quantity=1)`: lisää tuotteen, jos jo korissa → kasvattaa quantityä
- `removeItem(productId)`: poistaa tuotteen
- `updateQuantity(productId, quantity)`: asettaa määrän, ≤0 → poistaa
- `clearCart()`: tyhjentää korin
- `totalItems`: computed getter (sum of quantities)
- `totalPrice`: helper-funktio `getCartTotal(items, products)` (vaatii tuotteet parametrina)
- `persist` middleware → localStorage avaimella `webshop-cart`
- Ei `any`-tyyppejä
