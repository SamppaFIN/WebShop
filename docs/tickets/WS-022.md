# WS-022 · SEO ja metadata

**Vaihe:** 7 — Viimeistely | **Riippuvuudet:** WS-012 | **Arvioitu kesto:** 5 min

## Kuvaus
Open Graph -tagit, Twitter Cards, sitemap ja robots.txt.

## Hyväksymiskriteerit

1. `layout.tsx`: root metadata (title template, description, og:image, twitter:card, robots)
2. `products/[id]/page.tsx`: `generateMetadata()` dynaamisilla OG-tageilla
3. `src/app/sitemap.ts`: `sitemap()` generaattori — sis. staattiset sivut + kaikki tuotteet
4. `src/app/robots.ts`: `robots()` generaattori — sallii `/`, estää `/checkout`, `/orders`, `/account`
5. `.env.local`: `NEXT_PUBLIC_APP_URL=http://localhost:3000`

## Tiedostot

```
src/app/layout.tsx              ← päivitetään
src/app/products/[id]/page.tsx  ← päivitetään
src/app/sitemap.ts
src/app/robots.ts
.env.local                      ← NEXT_PUBLIC_APP_URL
```

## Verifiointi

```bash
curl http://localhost:3000/sitemap.xml
curl http://localhost:3000/robots.txt
```
