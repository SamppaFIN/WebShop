# WS-010 · CartDrawer-komponentti

**Vaihe:** 4 — UI | **Riippuvuudet:** WS-005, WS-007 | **Arvioitu kesto:** 8 min

## Kuvaus
Sivulta liukuva ostoskori (Sheet-komponentti). Listaa tuotteet määräsäädöillä.

## Hyväksymiskriteerit

`src/components/layout/CartDrawer.tsx` (Client Component):
- shadcn/ui `Sheet` oikealta, ohjattu `uiStore.isCartOpen`:lla
- Tyhjä: "Ostoskorisi on tyhjä" + linkki `/products`
- Tuotteet: pieni kuva, nimi (`truncate`), hinta, −/määrä/+ napit, poista (Trash2)
- Alapalkki: "Yhteensä: XX €", "Siirry kassalle" → `/checkout`, "Jatka ostoksia" → sulkee
- Framer Motion `AnimatePresence` tuotteiden lisäys/poisto-animaatioihin
- **Suositus:** `products: Product[]` propseina parentilta
