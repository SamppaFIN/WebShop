# WS-009 · Navbar-komponentti

**Vaihe:** 4 — UI | **Riippuvuudet:** WS-005, WS-006, WS-007 | **Arvioitu kesto:** 10 min

## Kuvaus
Kiinteä navigaatiopalkki: kori-badge, teema-toggle, käyttäjävalikko.

## Hyväksymiskriteerit

`src/components/layout/Navbar.tsx` (Client Component):
- `sticky top-0 z-50 backdrop-blur-md`
- Vasen: Logo "WebShop" → `/`, linkit "Etusivu", "Tuotteet"
- Oikea: ShoppingCart-ikoni + Badge (`totalItems`), teema-toggle (Moon/Sun), käyttäjä-avatar (DropdownMenu)
- Klikkaus koriin → `uiStore.openCart()`
- Kirjautumaton → "Kirjaudu"-nappi
- Mobile (≤768px): hamburger-menu (Sheet)
- Käyttää: `useCartStore`, `useThemeStore`, `useSession`
