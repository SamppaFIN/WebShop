# WS-023 · E2E-testit (Vitest + Playwright)

**Vaihe:** 7 — Viimeistely | **Riippuvuudet:** WS-005 | **Arvioitu kesto:** 12 min

## Kuvaus
Testaa cartStoren yksikkötestit ja checkout-päänäkymä Playwrightilla.

## Hyväksymiskriteerit

### Yksikkötestit (`src/__tests__/cartStore.test.ts` — Vitest):
- `addItem` lisää tuotteen
- `addItem` kahdesti kasvattaa quantityä
- `removeItem` poistaa tuotteen
- `updateQuantity` muuttaa määrän
- `updateQuantity` 0:aan poistaa tuotteen
- `clearCart` tyhjentää
- `totalItems` laskee oikein

### E2E-testi (`e2e/checkout.spec.ts` — Playwright):
- user → etusivu → klikkaa tuotekorttia → lisää koriin → kassa → ohjautuu loginiin

### Konfiguraatiot:
- `vitest.config.ts` (jsdom, globals, alias `@`)
- `playwright.config.ts`
- `.github/workflows/test.yml`
