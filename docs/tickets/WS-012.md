# WS-012 · App Router layout + sivupohjat

**Vaihe:** 5 — Sivut | **Riippuvuudet:** WS-009, WS-010, WS-011 | **Arvioitu kesto:** 8 min

## Kuvaus
Konfiguroi juurilayout ja luo kaikki reittisivut placeholder-versioina.

## Hyväksymiskriteerit

### `src/app/layout.tsx`
- ThemeProvider + SessionProvider + Navbar + CartDrawer + Footer
- Metadata: title template, description

### Sivut (placeholderit):
- `/` → `page.tsx`: HomePage
- `/products` → `products/page.tsx`: ProductsPage
- `/products/[id]` → `products/[id]/page.tsx`: ProductDetail
- `/cart` → `cart/page.tsx`: CartPage
- `/checkout` → `checkout/page.tsx`: CheckoutPage (suojattu)
- `/orders` → `orders/page.tsx`: OrdersPage (suojattu)
- `/login` → `login/page.tsx`: LoginPage
- `/account` → `account/page.tsx`: AccountPage (suojattu)
- `not-found.tsx`: 404-sivu
- `error.tsx`: yleinen virhesivu

## Verifiointi

```bash
npm run build && npm run dev
# Testaa jokainen reitti selaimessa
```
