# WS-019 · AccountPage

**Vaihe:** 5 — Sivut | **Riippuvuudet:** WS-012, WS-003 | **Arvioitu kesto:** 8 min

## Kuvaus
Käyttäjäprofiilin hallinta: tietojen muokkaus, osoitehallinta, uloskirjautuminen.

## Hyväksymiskriteerit

`src/app/account/page.tsx`:
1. Suojattu reitti
2. **Profiili:** avatar, nimi (muokattava), email (vain luku), "Tallenna"
3. **Osoitteet:** lista + "Lisää osoite" + muokkaa/poista + "Oletus"-merkintä
4. **Uloskirjautuminen:** destructive-nappi → `signOut()`

Server Actionit (`src/actions/user.ts`):
- `updateProfile`, `getAddresses`, `addAddress`, `updateAddress`, `deleteAddress`, `setDefaultAddress`
