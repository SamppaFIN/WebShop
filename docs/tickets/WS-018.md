# WS-018 · OrdersPage

**Vaihe:** 5 — Sivut | **Riippuvuudet:** WS-012, WS-004 | **Arvioitu kesto:** 6 min

## Kuvaus
Käyttäjän tilaushistoria statuksilla ja laajennettavilla yksityiskohdilla.

## Hyväksymiskriteerit

`src/app/orders/page.tsx`:
1. Suojattu reitti
2. Hakee `getMyOrders(userId)` → `Order[]`, uusin ensin
3. Tilauskortti: tilausnumero (#ORD-...), päivämäärä, summa (€), status Badge (värikoodattu)
4. Klikkaus laajentaa → tuotteet + toimitusosoite
5. Tyhjä tila: "Ei tilauksia vielä"
6. `?success=true` → toast "Tilaus vastaanotettu!"
