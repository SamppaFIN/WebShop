# WS-007 · shadcn/ui -komponenttien asennus

**Vaihe:** 4 — UI | **Riippuvuudet:** WS-000 | **Arvioitu kesto:** 5 min

## Kuvaus
Asenna tarvittavat shadcn/ui-komponentit: Button, Input, Badge, Card, Sheet, DropdownMenu, Avatar, Skeleton, Separator, Tabs.

## Hyväksymiskriteerit

Asennus: `npx shadcn@latest add button input badge card sheet dropdown-menu avatar skeleton separator tabs`

Komponentit `src/components/ui/`-hakemistossa: `button.tsx`, `input.tsx`, `badge.tsx`, `card.tsx`, `sheet.tsx`, `dropdown-menu.tsx`, `avatar.tsx`, `skeleton.tsx`, `separator.tsx`, `tabs.tsx`

## Verifiointi

```bash
ls src/components/ui/   # vähintään 10 komponenttia
npm run build           # ei virheitä
```
