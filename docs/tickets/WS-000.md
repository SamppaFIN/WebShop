## Kuvaus
Luo Next.js 15 -projekti TypeScriptillä, Tailwind CSS:llä ja shadcn/ui:lla. Konfiguroi Prisma, NextAuth, ympäristömuuttujat. Tämä on koko projektin perustus.

## Hyväksymiskriteerit

1. `npx create-next-app@latest webshop --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"` ajettu
   - HUOM: Koska `c:\Projects\WebShop` ei ole tyhjä, scaffoldaa väliaikaiseen kansioon ja siirrä.
2. Tailwind CSS v4 konfiguroitu `src/app/globals.css`:ssä: CSS-muuttujat `--color-primary`, `--color-secondary`, `--color-accent`, dark mode
3. shadcn/ui alustettu: `npx shadcn@latest init` (neutral base, CSS variables)
4. Prisma alustettu: `npx prisma init --datasource-provider postgresql`
5. NextAuth v5 asennettu: `npm install next-auth@beta @auth/prisma-adapter`
6. Asennukset: `stripe`, `@stripe/stripe-js`, `openai`, `zustand`, `zod`, `framer-motion`, `lucide-react`
7. `.env.local` sisältää kaikki ympäristömuuttujat placeholder-arvoilla
8. `.env.local` on `.gitignore`:ssa
9. `npm run build` menee läpi ilman virheitä

## Tiedostot

```
src/app/globals.css      ← Tailwind v4 + CSS-muuttujat
src/app/layout.tsx        ← juurilayout
src/auth.ts               ← NextAuth-konfiguraatio
prisma/schema.prisma      ← placeholder
.env.local                ← ympäristömuuttujat
components.json           ← shadcn/ui -konfiguraatio
```
