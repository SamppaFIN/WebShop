# WS-020 · Semanttinen haku (OpenAI embeddings + pgvector)

**Vaihe:** 6 — AI | **Riippuvuudet:** WS-001, WS-004 | **Arvioitu kesto:** 12 min

## Kuvaus
OpenAI-embeddingien generointi tuotteille ja semanttinen hakutoiminto pgvectorilla.

## Hyväksymiskriteerit

1. **pgvector:** `CREATE EXTENSION IF NOT EXISTS vector;` Supabasessa
2. **Prisma:** `Product.embedding Unsupported("vector(1536)")`
3. `src/lib/embeddings.ts`:
   - `generateEmbedding(text): Promise<number[]>` — OpenAI text-embedding-3-small
   - `cosineSimilarity(a, b): number`
   - `generateProductEmbeddingText(product): string` — yhdistää titlen, descriptionin, categoryn, tagit
   - `generateAndStoreEmbedding(productId): Promise<void>`
   - `generateAllEmbeddings(): Promise<void>`
4. `scripts/generate-embeddings.ts`: skripti, `npm run embeddings:generate`
5. `src/actions/products.ts` — `searchProductsSemantic(query)`: pgvector `<=>` cosine similarity
6. `src/app/api/products/search/route.ts`: `GET /api/products/search?q=hakusana`
7. Fallback: perinteinen ILIKE-haku jos ei OpenAI-avainta

## Tiedostot

```
src/lib/embeddings.ts
scripts/generate-embeddings.ts
src/actions/products.ts    ← päivitetään

```
