## Kuvaus
Määrittele koko tietokantaskeema Prismalla. Aja migraatio Supabase PostgreSQL:ään. Sisältää pgvector-laajennuksen. Ei sacred/consciousness-kenttiä.

## Hyväksymiskriteerit

1. `prisma/schema.prisma` sisältää mallit: `User`, `Account`, `Session`, `VerificationToken` (NextAuth), `Profile`, `Product`, `ProductImage`, `ProductVariant`, `Review`, `Cart`, `CartItem`, `Order`, `OrderItem`, `Address`, `SearchLog`
2. `Product.embedding` on tyyppiä `Unsupported("vector")` — pgvector
3. Ei sacred/consciousness/healing-kenttiä
4. `npx prisma migrate dev --name init` ajettu
5. pgvector-laajennus päällä Supabasessa: `CREATE EXTENSION IF NOT EXISTS vector;`
6. `prisma/seed.ts` sisältää 20 tuotetta 4+ kategoriassa, realistiset hinnat 5–500€
7. `npx prisma db seed` ajettu
8. `package.json`: `"prisma": { "seed": "tsx prisma/seed.ts" }`
