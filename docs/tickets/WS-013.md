# WS-013 · HomePage

**Vaihe:** 5 — Sivut | **Riippuvuudet:** WS-008, WS-012 | **Arvioitu kesto:** 10 min

## Kuvaus
Etusivu: hero-osio + featured-tuotteet + kategoriapikalinkit.

## Hyväksymiskriteerit

`src/app/page.tsx`:
1. **Hero:** otsikko "Löydä mitä tarvitset", alaotsikko, CTA "Selaa tuotteita" → `/products`, gradient-taustalla, Framer Motion fade-in
2. **Featured-tuotteet:** `getProducts({ limit: 4, sort: 'newest' })` → 4 ProductCardia, responsive grid (1/2/4 col), skeleton-lataus
3. **Kategoriapikalinkit:** `getCategories()` → 4-6 linkkiä `/products?category=X`, ikonit, responsive grid
4. Responsiivinen mobiilista 4K:hon
