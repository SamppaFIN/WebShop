# WS-006 · Theme store + UI store

**Vaihe:** 3 — State | **Riippuvuudet:** — | **Arvioitu kesto:** 6 min

## Kuvaus
Luo Zustand-storet teemalle (dark/light) ja UI-tilalle (cart drawer, mobile menu).

## Hyväksymiskriteerit

### `src/store/themeStore.ts`
- `theme: 'light' | 'dark'`, oletus `'light'`
- `toggleTheme()`: vaihtaa light ↔ dark
- `setTheme(theme)`: asettaa suoraan
- `persist` middleware → localStorage `webshop-theme`

### `src/store/uiStore.ts`
- `isCartOpen: boolean`, `isMobileMenuOpen: boolean`
- `openCart()`, `closeCart()`, `toggleCart()`
- `openMobileMenu()`, `closeMobileMenu()`, `toggleMobileMenu()`
- Ei persistointia (nollautuu refreshissä)

### `src/components/providers/ThemeProvider.tsx`
- Client-komponentti, käyttää `useThemeStore`
- `useEffect`: `document.documentElement.classList.toggle('dark', theme === 'dark')`
