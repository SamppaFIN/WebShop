# WS-008 · ProductCard-komponentti

**Vaihe:** 4 — UI | **Riippuvuudet:** WS-002, WS-007 | **Arvioitu kesto:** 8 min

## Kuvaus
Tuotekortti grid-näkymää varten. Framer Motion -hover-animaatio. shadcn/ui Card + Button.

## Hyväksymiskriteerit

`src/components/product/ProductCard.tsx`:
- Props: `product: Product`
- Näyttää: kuva (placeholder jos null), nimi (`line-clamp-2`), hinta (€), kategoria Badge
- `available: false` → "Loppunut" Badge (destructive)
- Klikkaus → `<Link href={/products/${product.id}}>`
- "Lisää koriin" -nappi → `cartStore.addItem(product.id)`, disabloitu jos `!available`
- Framer Motion: `whileHover={{ y: -4 }}`, `initial={{ opacity: 0, y: 20 }}`

`src/components/product/ProductCardSkeleton.tsx`:
- Skeleton-komponenteilla sama rakenne, `animate-pulse`
